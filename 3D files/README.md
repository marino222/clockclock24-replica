# 3D Models for ClockClock24 Replica

This directory contains all the 3D models used in this project. These models are essential for replicating the ClockClock24 design.

## Models Overview
- **Availability**: All models are hosted on [OnShape](https://www.onshape.com/).
- **Clock Hand Design**: The clock hand design was adapted from the [ClockClock24 Mechanical Replica](https://github.com/whosmatt/clockclock24-replica-mechanical/) repository by `whosmatt`.

## Usage
1. Access the models on OnShape.
2. Download or modify the models as needed for your project.
3. Integrate the models into your ClockClock24 replica build.

For more details, refer to the main [README](../README.md) in the project root.
