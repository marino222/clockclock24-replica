Firmware Update

ClockClock 24 (replica)

ℹ What file to upload:
Upload the compiled firmware binary .pio/build/esp32dev/firmware.bin.
To generate it, run pio run in the project's firmware/platformio/master folder. The file is created automatically by PlatformIO.
⚠ Before uploading:
Set the clock mode to OFF on the main page first. If the clock is running during the update the hands may end up out of sync and will need to be manually recalibrated after reboot.
⚠ If the update fails:
The device will keep running the current firmware. If the clock becomes unresponsive, cut power to reboot it, then recalibrate the hands manually.
